Spotify Song Recommender

The Spotify Song Recommender is an application designed to connect users with new music based off their existing music taste.

When given a song, the application will leverage Spotify's Search API and a NearestNeighbor Learning Model to recommend 5 new songs.

Done as an end of term project at the Bloom Institute of Technology, done with a class mate and two upperclassmen, our assignment was to apply some sort of tree model and utilize Spotify's API library to create an application hosted via Heroku.
